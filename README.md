# bif
Easily render any image into bloxd.\
Convert any arbitrary image into world code you can paste into your world.

---
You can
1. Paste your image into [the website](https://owcer.github.io/bif/converter/convert.html)
2. Copy world code
3. Paste world code in flat test world

> [!NOTE]
> bif stands for Bloxd Image Format

This is the best and fastest image converter in bloxd,
and it should take less than a minute to paste an image
and copy/paste the world code into a world

> [!WARNING]
> DO NOT use images that violate Bloxd rules.
